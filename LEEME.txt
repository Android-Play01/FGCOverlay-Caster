========================================================================
   FGC TOURNAMENT OVERLAY SYSTEM - v1.0
   Sistema de Marcadores para Torneos de Fighting Games
========================================================================

¡Gracias por usar este sistema de overlay! 
Este paquete contiene un Controlador (para el caster/admin) y un Arquitecto (el diseño visual para el stream) que se comunican en tiempo real.

------------------------------------------------------------------------
   REQUISITOS PREVIOS (¡IMPORTANTE!)
------------------------------------------------------------------------
Para que este sistema funcione, necesitas tener instalado Python en tu computadora. 
Esto permite crear el "puente" de comunicación entre el panel de control y el overlay.

1. Verifica si tienes Python: Abre una terminal (escribe 'cmd' en inicio) y escribe: python --version
2. Si te da error o no lo tienes, descárgalo gratis aquí: https://www.python.org/downloads/
   (Asegúrate de marcar la casilla "Add Python to PATH" durante la instalación).

------------------------------------------------------------------------
   PASO 1: INICIAR EL SISTEMA
------------------------------------------------------------------------
1. Dentro de esta carpeta, busca el archivo llamado "INICIAR_OVERLAY.bat".
2. Haz DOBLE CLIC sobre él.
3. Se abrirá una ventana negra (consola).
   >>> ¡NO CIERRES ESTA VENTANA! <<<
   Mientras esa ventana esté abierta, el sistema estará funcionando. Si la cierras, el overlay dejará de responder.

------------------------------------------------------------------------
   PASO 2: CONFIGURACIÓN EN OBS STUDIO
------------------------------------------------------------------------

A) AGREGAR EL CONTROLADOR (DOCK):
   1. En OBS, ve al menú superior: Vista > Paneles (Docks) > Paneles de navegador personalizados...
   2. En "Nombre del panel", escribe: Controlador FGC
   3. En "URL", escribe exactamente esto:
      http://localhost:8000/controlador.html
   4. Dale a "Aplicar". Aparecerá el panel, puedes anclarlo donde quieras en tu OBS.

B) AGREGAR EL OVERLAY (DISEÑO):
   1. En tu escena, agrega una nueva fuente: Fuente de Navegador (Browser Source).
   2. Ponle de nombre: Overlay FGC
   3. Desmarca la casilla "Archivo local".
   4. En "URL", escribe exactamente esto:
      http://localhost:8000/arquitecto.html
   5. En Ancho (Width): 1920
   6. En Alto (Height): 1080
   7. Dale a "Aceptar".

------------------------------------------------------------------------
   SOLUCIÓN DE PROBLEMAS
------------------------------------------------------------------------
- ¿El controlador no cambia el overlay?
  Asegúrate de haber puesto las URLs como http://localhost:8000/... y NO como archivos locales (C:/...).

- ¿Se cerró la ventana negra sola?
  Probablemente no tengas Python instalado correctamente. Revisa el paso de "Requisitos".
